# Example ai.json Files

This directory contains example `ai.json` configurations demonstrating various features and use cases.

## Available Examples

### [superpowers](./superpowers/)

A comprehensive example based on [obra/superpowers](https://github.com/obra/superpowers), demonstrating:

- **14 skills** covering testing, debugging, collaboration, and meta workflows
- **3 prompts** for brainstorming, planning, and execution
- **1 rule** for the superpowers introduction
- **Hooks** for session initialization (works across Claude Code, Windsurf, and Cursor)

## Installation

You can install any example directly using the raw GitHub URL:

```bash
# Install the superpowers example
aix install https://raw.githubusercontent.com/a1st/aix/main/examples/ai-json/superpowers/ai.json
```

Or copy the `ai.json` file to your project and customize it.

## Creating Your Own

Use these examples as templates for your own configurations. See the [ai.json schema](../../packages/schema/schema.json) for all available options.
