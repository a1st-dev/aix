description: Install aix globally to manage AI editor configuration.
---

## Prerequisites

- Node.js 20.19.0+ or 22.12.0+
- npm 10.0.0+

## Install

```bash
npm install -g @a1st/aix
```

## Verify

```bash
aix --version
```

You should see the installed version number.

## Next steps

Continue to the [Quick Start](/getting-started/quick-start/) guide to create your first `ai.json` configuration.
