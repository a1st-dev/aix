---
title: Commands Overview
description: Summary of all aix CLI commands.
---

| Command | Description |
|---------|-------------|
| [`init`](/commands/init/) | Create an ai.json file |
| [`add`](/commands/add/) | Add skills, rules, prompts, or MCP servers |
| [`install`](/commands/install/) | Install config to an editor |
| [`search`](/commands/search/) | Search the MCP registry |
| [`list`](/commands/list/) | List configured items |

## Global options

All commands support these options:

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help for the command |
| `--version`, `-v` | Show version number |
